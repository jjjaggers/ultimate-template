FAQs

How do I rebrand this quickly?

Edit the CSS variables in :root (colors, spacing, type scale). Replace logo text and hero copy.

Can I remove sections?

Yes—delete the section block or comment it out. The layout is modular.

Is there any JavaScript?

No. The hamburger uses a CSS checkbox toggle. The FAQ uses <details>.
